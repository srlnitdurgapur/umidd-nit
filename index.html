Installation Guide


  • Install Python version 3.9.
  • Install an IDE (Integrated Development Environment) such as Jupyter Notebook or VS Code, etc.
  • Create a conda environment (using conda create -n myenv python=3.9) or create one via Anaconda Navigator.
  • Install Python libraries: mediapipe, tensorflow, joblib, scikit-learn, opencv-python, matplotlib, numpy.
  • Set the location of the respective deep learning and machine learning models in the code segment.
  • If using an integrated camera, set the camera index to 0; use index 1 for a secondary camera.
  • Execute the code and demonstrate various hand gestures to observe different actions without manual intervention.
